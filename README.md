# hoai-qrcode
This repsoitory is a static webpage for generating a QR Code from a URL or one line of text.
There is a box for inputting the text and the generated QR Code is shown below. A button, underneath, lets you download the code as a PNG image. This code in generated by an AI agent, as an example of how to use it.



Testing
========

Serve this folder with a simple HTTP server and open the page for testing. 
This repository is deployed as a static webpage using the default GitHub page workflow.
